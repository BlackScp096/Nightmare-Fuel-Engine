# Nightmare Fuel Engine
A custom Friday Night Funkin' engine created by BlackScp096.

This starter pack includes a ready-to-edit `source/` folder with core systems:
- Logging and crash handling
- Psych-compatible mod loading hooks
- Inappropriate content scanner (warning only)
- Ghost tap support (4-key gameplay)
- MP4 cutscene placeholder support
- GameJolt trophy stubs

## How to use
1. Install Haxe, haxelib, Lime, OpenFL, Flixel.
2. Open the folder in your editor (Visual Studio Code recommended).
3. Edit source files in `source/` and add assets in `assets/`.
4. Build with `lime test windows -debug` when ready.
